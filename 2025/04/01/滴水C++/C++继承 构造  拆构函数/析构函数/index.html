析构函数

/*析构函数：
1. 做收尾的，主要用于清理工作
2.析构函数不允许重载，它只有一个
3.不能带参数
4.不能带返回值
5.编译器不要求必须提供

/
//—-栈 全局变量 堆
struct person
{
int age;
int level;
char arr;

person()
{
    this->age = age;
    this->level = level;
    arr= (char*)malloc(1024);		//在堆中申请空间  要自己对堆里的东西释放
}

//—–对象不用了，要释放
~person()
{
free(arr);
}
};

new+delete

new: nh_malloc_dbg _heap_alloc_dbg _heap_alloc_base HeapAlloc
delete free _free_dbg _free_base HeapFree

malloc nh_malloc_dbg _heap_alloc_dbg _heap_alloc_base HeapAlloc
ffree _free _free_dbg _free_base HeapFree

在C++里 new和malloc调用的东西一样				//底层调用一样

int* pi = new int(5); //在堆里创建一个int，初始值是5.

int* i = new int[5]; //创建了5个int，即数组
delete[5] i; //new的时候，多个，那就要加[ ]释放.

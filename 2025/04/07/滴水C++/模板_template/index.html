模板_template

顺序查找（没排序）
折半查找（已排序） 找中间的数比较

位运算 (x+y)<<1 相当于乘以二
（x+y)>>1 相当于除以二

模板的作用：对于一些不同类型的数，计算机来替我们进行复制，简短任务和代码量
代码的赋值

template //需要变量的数，替换掉就行了
void sort(T arr,int nlength)

template<class T，class E>
void sort(T arr,E nlength)

对结构体/类中使用
template<class T，class M>
struct Base
{
T x；
T y;
M a;
M b;
}
